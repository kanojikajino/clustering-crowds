# -*- coding: utf-8 -*-
Python Implementation of the CPC method that appeared in "Clustering Crowds".
(i) Create an object of the class "SparseData" using crowd-generated training data,
(ii) Create an object of the class "CPCMethod" using SparseData, 
(iii) and use the function optimize() to get the estimations.

Documents can be obtained by using the following commands:
epydoc *.py
